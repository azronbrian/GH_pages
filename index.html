Using GitHub pages to host static site.

GitHub can not only store your profiles remotely but also host your project. This happens freely for public repositories. If your repo is private, you'll need to upgrade in order to host your project.

Steps

  1. Initialize git init in your project files directory
  2. Add all the projects files
  3. Commit them using suitable messages
  4. Create a new repo in your GitHub account
  5. Push the existing repo from your computer to the newly created Github repo
  6. Go to GitHub pages settings and select branch and root directory then save.
  7. You will be provided with a live link for your hosted project.
